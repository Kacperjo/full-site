Witam na fancy str.

Na tej str wytłumacze troche css.

To będzie tak wyglądać:
  • zobaczysz pudełko div lub zielony tekst
  • bedzie miało efekty(czasem po nakierowani wskaźnikiem)
  • pod tym będzie napisany element html i css którym zrobiono ten efekt
ruch w bok po najechaniu

html:

<div class="side-move div-look"> ruch w bok po najechaniu </div>

css:

.div-look { border: 4px solid red; border-radius: 4px; background-color: pink; width: 250px; }

.side-move { margin-left: 2px; transition: margin 0.5s ease; }

.side-move:hover { margin-left: 12px; }

rotacjia

html:

<div class="div-look rotate"> rotacjia </div>

css:

.rotate {
width: 100px;
height: 100px;
margin-left: 10px;
font-size: 15px;
text-align: center;

}

.rotate:hover {

}
